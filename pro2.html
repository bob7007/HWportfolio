Temperature, Light and Touch Sensor



Previous Next


The project consists of reading analog values into the microcontroller “PSP430”, convert them to digital using its internal ADC and finally processing the data using a C programming language.

The temperature sensor LM34DZ a touch switch and a photo cell are connected to the microcontroler which takes initial baseline readings and stores them in corresponding variables.

The sensors are also connected externally to an LED which will light when the values read are outside the programmed range previously determined by the baseline

Finally, while running the program on an infinite loop, input data can be changed in real time through the sensors which trigger the LED’s when the values are outside the specified range.
