BCD to 7 Segment LED Display



Previous Next


The parallel loading counter is connected to a double 7 segment display using a 2-input input selector and 2 op amps acting as oscillators to provide the clock pulse signal.

Switches directly connected to the BCD control the direction of the count and allow for specific input loading.

Another switch allows for automatic counting through a pulse signal created by two operational amplifiers (op amp). The signal frequency is set by adjusting the resistivity of the RC circuit. A slow signal is created to manage the speed of the count, likewise the fast signal determines the duty cycle or the ammount of time the LED display will be on which determines its brightness.
